# Vectors

- `conformance/` — primitive-level vectors asserting Mnemosyne's vendored canonical-core is
  byte-identical to paradigm_terra (D1, anti-drift). Generated by porting terra's
  `canonical/vectors/golden.json` for the shared primitives + the four CE_V13 Merkle/state tags.
- `spine/` — golden vectors for the deterministic spine (`append` outputs: object_id, space_state,
  vault_memory_root) over a fixed, seeded scenario. Status NORMATIVE (promoted at v1.0.0): the cross-impl contract over deterministic reference scenarios.
